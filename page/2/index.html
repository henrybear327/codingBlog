Drying

Solution sketch

別懶著推公式呀！QQ 二分搜尋被坑過幾次了還是被繼續坑QQ

設 x 為烘衣服時間：
$k * x + (mid - x) >= a_i$
$x(k - 1) >= a_i - mid$
$x >= \frac{(a_i - mid)}{(k - 1)}$

注意 special case!!
